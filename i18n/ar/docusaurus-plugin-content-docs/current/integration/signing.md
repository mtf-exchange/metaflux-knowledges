# جولة التوقيع

:::info
**انتقلت هذه الصفحة.** يتم توقيع إجراءات `/exchange` باستخدام **بيانات مكتوبة منظَّمة وفق EIP-712** (`eth_signTypedData_v4`). هذا هو نظام التوقيع الوحيد المعتمد.
الجولة الشاملة — النطاق، وسلاسل الأنواع لكل إجراء، والملخص الرقمي، والأمثلة التطبيقية، والتحقق المحلي — متاحة الآن في
[**توقيع البيانات المكتوبة**](./typed-data-signing.md).
:::

كل طلب `/exchange` هو توقيع بيانات مكتوبة وفق EIP-712: تعرض المحفظة كل حقل من حقول الإجراء باسمه، ويُعيد الخادم بناء البنية المكتوبة من `action.type` و`action.params`، ويعيد حساب الملخص الرقمي، ويستخرج هوية الموقِّع (الحساب، أو [وكيل](../concepts/agent-wallets.md) معتمد تابع له). لا يوجد نظام ثانٍ للاختيار بينهما.

اذهب إلى [**توقيع البيانات المكتوبة**](./typed-data-signing.md) للاطلاع على المواصفات الكاملة وأمثلة TypeScript / Python جاهزة للنسخ والاستخدام.
