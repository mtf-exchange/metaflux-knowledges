## Devnet

بيئة الاختبار التكاملي. يمكنك الحصول على USDC مجانًا عبر الصنبور؛ والحالة مؤقتة (تُعاد تهيئتها بشكل دوري).

البوابة هي نقطة الدخول العامة الوحيدة. يُخدَّم السطح الأصلي لـ MTF على
`/info` · `/exchange` · `/ws`؛ وواجهة EVM JSON-RPC على `/evm`.

| الخدمة | نقطة النهاية |
|--------|-------------|
| البوابة الرئيسية | `https://api.devnet.mtf.exchange` |
| MTF-native | `POST /info` · `POST /exchange` · `GET /ws` |
| EVM JSON-RPC | `POST /evm` |
| الصنبور (devnet/testnet) | `POST /faucet` |
| Gateway WS (native) | `wss://api.devnet.mtf.exchange/ws` |
| المستكشف | `https://app.mtf.exchange/explorer` |
| الحالة | `https://status.mtf.exchange/devnet` |

هل تشغّل العقدة بنفسك؟ تخدم العقدة الواجهة الأصلية ذاتها مباشرةً على
`http://localhost:8080` (`/info` · `/exchange` · `/ws` · `/faucet`)، وواجهة EVM RPC الخام على `http://localhost:8545`. هذه منافذ الاستضافة الذاتية وليست عناوين URL عامة.

| معاملات التوقيع | القيمة |
|----------------|--------|
| `chainId` | `31337` |
| اسم نطاق EIP-712 `name` | `"MetaFlux"` |
| إصدار نطاق EIP-712 `version` | `"1"` |
| عقد التحقق في نطاق EIP-712 `verifyingContract` | `0x0000000000000000000000000000000000000000` |

ربط USDC الجسري: عبر **جسر الحضانة MetaBridge** ([الجسر](./bridge/))، لا عبر Circle CCTP. تستخدم الإيداعات في بيئة الاختبار نشر `MetaBridgeUSDC` على Base Sepolia إلى جانب USDC الاختباري من Circle على Base Sepolia.

### الصنبور

يُضيف `POST /faucet` على البوابة الرئيسية أموالًا اختبارية إلى عنوان معين.
متاح على Devnet/Testnet فقط — هذا المسار **لا يُثبَّت أبدًا على الشبكة الرئيسية** (`chainId 8964`).
المنحة بحالة **`"queued"`** — مُدرَجة في انتظار الكتلة التالية، لذا يتحدث الرصيد
بعد ~كتلة واحدة تقريبًا، وليس بشكل فوري. المرجع الكامل: [`POST /faucet`](api/rest/faucet.md).

```bash
curl -X POST https://api.devnet.mtf.exchange/faucet \
  -H 'content-type: application/json' \
  -d '{"address":"0x<YOUR_ADDRESS>"}'
# -> {"address":"0x…","usdc":3000,"mtf":10,"status":"queued"}
```

- تمنح **3000 USDC** كضمان مشترك **+ 10 MTF** فوري — **مرة واحدة فقط لكل
  عنوان** (الطلب الثاني يُرجع `429 address already funded`).
- `amount` اختياري (USDC صحيح)؛ يُحدِّد منحة USDC بحدٍّ أقصى *تنازلي* (≤ 3000). MTF ثابت.
- محدود بمعدل طلب واحد / دقيقة / IP (يُرجع `429` عند التجاوز).
- `400` عنوان غير صالح · `429` تم التمويل مسبقًا / محدود بالـ IP · `503` قائمة الانتظار ممتلئة — الجسم `{"error":"…"}`.

### إعادة تهيئة الحالة

قد تُعاد تهيئة Devnet عند ترقيات البروتوكول. الجدول الزمني: حسب الحاجة خلال مرحلة التطوير قبل الإطلاق الرئيسي؛ مع إشعار أسبوعي كلما أمكن ذلك. تابع [صفحة الحالة](https://status.mtf.exchange/devnet) للاطلاع على إعلانات إعادة التهيئة.

## Testnet (مخطط له)

شبكة التدريب قبل الإطلاق الرئيسي، مع ضمانات استقرار.

| الخدمة | نقطة النهاية |
|--------|-------------|
| Gateway REST | TBD |
| Gateway WS | TBD |
| الصنبور | TBD (محدود المعدل) |
| المستكشف | TBD |

معرِّف سلسلة Testnet: `114514` (`0x1bf52`). MetaFlux شبكة مستقلة ذات معرِّفات سلسلة خاصة بها.

الفوارق بين Testnet والشبكة الرئيسية:
- يُربط USDC عبر MetaBridge من سلسلة مصدر اختبارية (USDC الاختباري على Base Sepolia)، وليس USDC حقيقيًا.
- مجموعة المحققين يتحكم فيها المشغّل.
- لا قيمة اقتصادية حقيقية.

هيكل اتصال Testnet مطابق تمامًا للشبكة الرئيسية. يجب أن تتطلب العملاء المختبرة على Testnet **تغيير `chainId` وعنوان URL الأساسي فقط** للتحويل إلى الشبكة الرئيسية.

الفوارق بين Mainnet وDevnet/Testnet:

- مجموعة المحققين غير مقيّدة (منتخبة عبر الحوكمة).
- قيمة اقتصادية حقيقية.
- حدود المعدل والرسوم وفقًا لـ [حدود المعدل](./api/rate-limits.md) و[الرسوم](./concepts/fees.md).

## ممرات الجسر

تُربط USDC (والأصول الأخرى) عبر **جسر الحضانة MetaBridge** — توقيع مشترك بوزن ثلثَي الحصة بين المحققين، دون اعتماد على Circle CCTP. سلاسل المصدر:

| السلسلة | الحالة |
|---------|--------|
| Base | **مباشر على Base Sepolia** (`MetaBridgeUSDC` (MetaBridgeAlpha) [`0xA6c914Cd59F8B3A8551B5f24b047d78542063a00`](https://sepolia.basescan.org/address/0xA6c914Cd59F8B3A8551B5f24b047d78542063a00))؛ الشبكة الرئيسية قيد المراجعة الأمنية |
| Solana | **مباشر على devnet** (برنامج `metabridge-solana` [`8nahcGhCtXpsZ31mHmHinCRf5MX1qWQzruMj6E1KMCwi`](https://solscan.io/account/8nahcGhCtXpsZ31mHmHinCRf5MX1qWQzruMj6E1KMCwi?cluster=devnet))؛ الشبكة الرئيسية قيد المراجعة الأمنية |
| Arbitrum | مخطط له |

راجع [الجسر](./bridge/) للاطلاع على تدفق الإيداع والسحب وجدول النشر.
