## Соглашения

- Описанные здесь эндпоинты представляют **стабильный, публичный** интерфейс обмена данными.
- Примеры запросов и ответов используют реальные форматы — их можно копировать напрямую.
- Поля цены и размера представлены как целые числа с фиксированной запятой (8 знаков после запятой); суммы в USDC — в базовых единицах с 6 знаками. Оба типа передаются как JSON-строки во избежание потери точности по стандарту IEEE-754.
- Все временные метки в полях `_ts` / `_ms` — это миллисекунды Unix (получены из консенсуса).

## Статусы документации

Каждый документ содержит тег «Статус» в начале:

- **stable** — формат данных зафиксирован для V1; можно безопасно использовать в разработке.
- **preview** — работает уже сейчас; незначительные изменения формата возможны до выхода в мейннет (отмечены явно).
- **planned** — описано, но ещё не реализовано.

Смотрите раздел [версионирование](./versioning.md) для ознакомления с политикой управления изменениями.
