# Справочник API

Единый нативный протокол MTF, обслуживаемый единой точкой входа шлюза
(`https://<net>-gateway.mtf.exchange`).

| Поверхность | Где | Примечания |
|-------------|-----|------------|
| **MTF-native** | `POST /exchange`, `POST /info`, `GET /ws`, `POST /faucet` | Компактный формат snake_case. Открывает полный доступ, включая расширенные функции MTF (RFQ, FBA, подключение к PM, кросс-чейн). |

> Шлюз — точка входа для нативной поверхности MTF
> (`/info`, `/exchange`, `/ws`). Запускаете ноду самостоятельно? Она предоставляет ту же
> нативную поверхность напрямую по адресу `http://localhost:8080`.
