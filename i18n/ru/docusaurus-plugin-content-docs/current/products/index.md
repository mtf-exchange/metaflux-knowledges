# Продукты

Обзор **торговых продуктов**, доступных на MetaFlux. Каждый из них представляет собой отдельный тип рынка со своим стаканом заявок, балансами и моделью управления рисками. В этом разделе описано, что собой представляет каждый продукт, его текущий статус и ссылки на подробную механику. Для ознакомления с общими механизмами — типами ордеров, маржой, ликвидацией, комиссиями — см. раздел [Концепции](../concepts/index.md).

## Доступные инструменты

| Продукт | Описание | Статус |
|---|---|---|
| [Бессрочные контракты](./perpetuals.md) | Торговля с кредитным плечом в лонг/шорт по цене актива без срока экспирации, привязанная ставкой финансирования | **В работе** |
| [Спот](./spot.md) | CLOB с прямым обменом токенов, расчёты по балансу, без кредитного плеча | **В работе** |
| [Маржинальный спот](./spot-margin.md) | Спот с кредитным плечом, финансируемый из пула кредитования [Earn](../concepts/earn.md) | **Превью на Devnet** |
| [Опционы](./options.md) | Опционы на блокчейне (колл / пут) | **Запланировано** |
| [CDS](./cds.md) | Контракты защиты от кредитного дефолта | **Запланировано** |

**Бессрочные контракты** — основной рынок платформы и главное место для торговли с кредитным плечом: ставки финансирования, маркировочные цены, режимы маржи и механизм ступенчатой ликвидации описаны в первую очередь применительно к перпам, если не указано иное. **Спот** — базовый режим без плеча, только с расчётами по балансу. **Маржинальный спот** — опциональный режим с кредитным плечом на споте, где пул [Earn](../concepts/earn.md) выступает стороной, предоставляющей ликвидность для кредитования. **Опционы** и **CDS** находятся в стадии планирования; актуальное состояние проекта см. на соответствующих страницах.

:::info
**Только беспроцентный спот соответствует нормам шариата.** Из всех перечисленных продуктов лишь **безрычажная** [спот-торговля](./spot.md) — покупка и продажа по полной стоимости **без кредитного плеча, без маржи, без заимствований и без ставки финансирования** — в целом признаётся совместимой с принципами исламских финансов (шариата). К несовместимым продуктам явно относятся **маржинальный спот (спот с кредитным плечом)**, **бессрочные фьючерсы** и любые другие инструменты с плечом или деривативной структурой — кредитное плечо и заимствования вводят элементы процента (риба), спекуляции и неопределённости (майсир, гарар). Данная информация носит ознакомительный характер и не является религиозным или финансовым советом.
:::
