# Пошаговое руководство по подписанию

:::info
**Эта страница перемещена.** Действия `/exchange` подписываются с помощью **структурированных типизированных данных EIP-712** (`eth_signTypedData_v4`). Это единственная схема подписания.
Полное пошаговое руководство — домен, строки типов для каждого действия, дайджест, разобранные примеры и локальная верификация — теперь находится в разделе
[**подписание типизированных данных**](./typed-data-signing.md).
:::

Каждый запрос `/exchange` представляет собой подпись типизированных данных EIP-712: кошелёк отображает каждое поле действия по имени, сервер восстанавливает типизированную структуру из `action.type` + `action.params`, пересчитывает дайджест и извлекает подписанта (аккаунт или утверждённый [агент](../concepts/agent-wallets.md) этого аккаунта). Никакой альтернативной схемы для выбора не существует.

Перейдите к разделу [**подписание типизированных данных**](./typed-data-signing.md) для получения полной спецификации и готовых к использованию примеров на TypeScript / Python.
