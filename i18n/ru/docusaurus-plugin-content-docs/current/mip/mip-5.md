# MIP-5 — Earn

:::info
**Запланировано.** MIP-5 отводит ранее зарезервированный слот под **Earn** — пул кредитования, в котором вкладчики предоставляют активы и получают доход от процентов, уплачиваемых заёмщиками спот-маржи. Спецификация и запуск следуют за реализацией спот-маржи, на которую опирается данный модуль.
:::

## Что такое Earn

Earn — это **сторона предложения на спот-рынке кредитования MetaFlux**. Вкладчик размещает актив (например, USDC) в пул конкретного актива и получает **доли** (shares), оценённые по стоимости чистых активов (NAV) пула — та же схема учёта NAV/доля, что и в
[хранилище Metaliquidity](./mip-2.md). Трейдеры спот-маржи занимают средства из пула и выплачивают проценты; эти проценты накапливаются в NAV пула, увеличивая стоимость каждой доли. Доход вкладчика рассчитывается по формуле:

```
your_yield = shares × (share_price_now − share_price_at_deposit)
```

Ставка заимствования является детерминированной функцией **утилизации** (занято ÷ предложено). APY предложения следует за ней:

```
supply_APY ≈ borrow_APR × utilisation × (1 − reserve_factor)
```

## Две стороны одного рынка

Earn (предложение) и спот-маржа (спрос) — это две стороны единого рынка кредитования: проценты, которые платят заёмщики, **и есть** доход, который получают кредиторы.

- **Повторно использует** учёт NAV/доля из [MIP-2](./mip-2.md) для пополнения и вывода средств (пополнение выпускает доли по текущей цене доли; вывод погашает их по NAV).
- Добавляет то, чего нет в хранилище: кривую процентной ставки «утилизация → APR» и непрерывное (поблочное) начисление процентов.
