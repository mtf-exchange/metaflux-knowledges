# MIP-6 — Рынки исходов / предсказаний

:::info
**Отложено до V3.** Не входит в область охвата v1 или v2. Перенумеровано из MIP-4.
:::

MIP-6 — это механизм MetaFlux для **рынков исходов / предсказаний**: on-chain рынки, на которых пользователи торгуют на основе разрешения бинарных или категориальных исходов (аналог предложения по улучшению рынков предсказаний на устоявшихся on-chain площадках). Это перспективное направление дифференциации, отложенное до **V3**.

## Почему это выделено в отдельный номер

Рынки исходов **изначально имели номер MIP-4**. Когда проект переориентировал MIP-4 на
[агрегатор ликвидности / интернализатор для перпов](./mip-4.md), концепция
рынков исходов была перенумерована в **MIP-6** и перенесена из бэклога V2 в бэклог V3.
Присвоение нового номера позволяет избежать путаницы, которая возникла бы при повторном использовании MIP-4 для двух несвязанных механизмов. Не следует называть рынки исходов «MIP-4».

## Почему отложено

- Это направление дифференциации имеет более низкий приоритет: деривативы / бессрочные контракты остаются основным полем конкурентной борьбы, а розничная выручка от агрегатора MIP-4 значительно превосходит потенциал рынков исходов.
- Расчёт по рынкам исходов добавляет операционную сложность клиринга, которая в противном случае не нужна ядру протокола: он зависит от разрешения внешних оракулов, временных окон и механизмов оспаривания.
- Рынки предсказаний несут юрисдикционно-специфическую регуляторную чувствительность, которую лучше решать после того, как базовый протокол достигнет зрелости.

Когда рынки исходов будут выпущены, они выйдут под номером MIP-6 с собственным дизайном разрешения / оракулов / оспаривания — ни один из этих компонентов сегодня не резервируется заранее.
