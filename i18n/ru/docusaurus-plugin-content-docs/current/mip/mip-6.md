# MIP-6 — Рынки исходов / прогнозирующие рынки

:::info
**Отложено до V3.** Не входит в область действия v1 или v2. Перенумерован из MIP-4.
:::

MIP-6 — это механизм MetaFlux для **рынков исходов / прогнозирующих рынков** — торговых площадок на блокчейне, где пользователи торгуют на основе разрешения бинарных или категориальных исходов (аналог предложений по улучшению прогнозирующих рынков на устоявшихся on-chain платформах). Это будущая возможность, отложенная до **V3**.

## Почему этому присвоен отдельный номер

Рынки исходов **изначально имели номер MIP-4**. Когда проект перепрофилировал MIP-4 в
[агрегатор ликвидности / интернализатор для перпов](./mip-4.md), концепция
рынков исходов была перенумерована в **MIP-6** и перенесена из V2 в бэклог V3.
Присвоение нового номера позволяет избежать путаницы, которая возникла бы при повторном использовании MIP-4 для двух несвязанных механизмов. Не следует называть рынки исходов «MIP-4».

## Почему отложено

- Это направление имеет более низкий приоритет: деривативы / бессрочные контракты остаются главным полем конкурентной борьбы, а розничная выручка от агрегатора MIP-4 значительно превосходит потенциал рынков исходов.
- Расчёт по рынкам исходов добавляет клиринговую сложность, которая в остальном ядру не нужна: он зависит от разрешения внешних оракулов, временных окон и механизмов оспаривания.
- Прогнозирующие рынки несут специфическую для разных юрисдикций регуляторную чувствительность, которую лучше рассматривать после того, как базовый протокол достигнет зрелости.

Когда рынки исходов будут выпущены, они выйдут как MIP-6 с собственным дизайном разрешения / оракулов / оспаривания — ни один из этих компонентов сегодня не резервируется заблаговременно.
