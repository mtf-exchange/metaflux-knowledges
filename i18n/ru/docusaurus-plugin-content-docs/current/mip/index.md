# MIP — Предложения по улучшению рынка

:::info
**Статус.** MIP-1 реализован · MIP-2 в работе · MIP-3 реализован · MIP-4 запланирован (V2) · MIP-5 (Earn) запланирован · MIP-6 отложен (V3).
:::

MetaFlux использует нумерованную модель предложений по улучшению (аналогичную схемам, принятым в известных on-chain протоколах бессрочных контрактов) для изменений на уровне протокола, затрагивающих котируемые рынки, нативную ликвидность или основные механизмы комиссий.

| MIP | Название | Статус |
|-----|----------|--------|
| [MIP-1](./mip-1.md) | Стандарт спот-токена + развёртывание рынка | Реализован |
| [MIP-2](./mip-2.md) | Metaliquidity — протокольное хранилище ликвидности | В работе |
| [MIP-3](./mip-3.md) | Безразрешительное развёртывание рынка бессрочных контрактов | Реализован |
| [MIP-4](./mip-4.md) | Агрегатор / интернализатор ликвидности бессрочных контрактов | Запланирован (V2) |
| [MIP-5](./mip-5.md) | Earn — пул спот-кредитования | Запланирован |
| [MIP-6](./mip-6.md) | Рынки исходов / предсказаний | Отложен (V3) |

Предложения по развёртыванию разделяют спот и бессрочные контракты так же, как это принято на известных площадках: **MIP-1** — безразрешительное развёртывание спот-токена и рынка (семейство действий `spotDeploy`), **MIP-3** — безразрешительное развёртывание рынков бессрочных контрактов силами сторонних разработчиков (семейство действий `perpDeploy`). Оба механизма работают на базе одних и тех же трёх потоков газовых аукционов. (В текущей реализации оба семейства действий по-прежнему объединены в одном модуле, а спот-путь помечен как «MIP-3»; это несоответствие устраняется — поведение остаётся неизменным.) **MIP-2 (Metaliquidity)** — хранилище нативной ликвидности, принадлежащей протоколу. **MIP-4** — агрегатор под управлением MetaFlux, обрабатывающий розничный поток и дополняющий безразрешительные рынки. **MIP-6** (Рынки исходов) ранее имел номер MIP-4 и был перенумерован, когда MIP-4 переопределили как агрегатор. **MIP-5 (Earn)** — сторона предложения пула кредитования: вкладчики получают доход из процентов, которые платят заёмщики спот-маржи; используется модель NAV/доля из MIP-2.

## Объём V1

V1 охватывает MIP-1, MIP-2 и MIP-3. MIP-4 (агрегатор) запланирован на V2; MIP-6 (Рынки исходов) отложен до V3. MIP-5 (Earn) запланирован и будет реализован после завершения работ по спот-марже, на которых он основан.
