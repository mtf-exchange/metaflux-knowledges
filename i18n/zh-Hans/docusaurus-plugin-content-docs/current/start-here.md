# MetaFlux 101 — 从这里开始

:::info
**初次接触MetaFlux？** 本页面不假设你对加密货币或衍生品有任何先前的了解。读完这一页，你将理解MetaFlux是什么、你现在能在上面做什么，以及开始第一笔交易前需要知道的几个概念。
:::

## MetaFlux是什么

MetaFlux是一个**开放的、在链上的交易所** — 一个交易市场，完全运行在公共网络上，而不是在某一家公司的私有服务器内。可以把传统交易所看作是一栋楼，有一个上锁的后办公室：你必须相信握着钥匙的人。MetaFlux则更像是一个公共账本，任何人都可以查看，也没有任何单一方能控制它。规则写在代码里，每笔交易都公开记录，你保管自己的资金。

因为MetaFlux运行在链上，它是**透明的**（任何人都可以验证发生了什么），**开放的**（任何人都可以接入、基于它进行开发或挂牌交易市场），**始终在线的**（它由许多独立运营者运行，而不是一家可以关闭的公司）。你用加密钱包连接、存入资金、交易 — 无需账户申请，无需把关人。

## 你现在能做什么

- **交易永续期货。** 对资产价格是上升还是下降进行头寸操作，可选杠杆，而无需持有该资产本身。（关于"永续"的含义，见下文。）
- **现货交易。** 买卖资产本身，用你的余额进行结算（仅限余额 — 暂无杠杆）。
- **持有双向（套期保值）头寸** — 在同一市场同一时间保持多头和空头头寸。见 [对冲模式](concepts/hedge-mode.md)。

即将推出：

- **通过 [Earn](concepts/earn.md)（一个支付利息的借贷池）赚取闲置USDC的收益**。
- **杠杆现货交易**（现货保证金），将与Earn一起推出。见 [现货保证金](concepts/spot-margin.md)。

## 需要了解的几个关键概念

你不需要成为专家才能开始，但理解几个概念会让其他一切都变得清晰。每个概念都链接到更详细的解释。

**永续vs.现货。** *现货*是最直接的交易方式：你用一种资产换另一种资产并拥有结果。*永续期货*（"perp"）是一种追踪资产价格的合约，让你能从其价格变动中获利 — 无论上升还是下降 — 而无需拥有该资产，且没有到期日期，所以只要你保持头寸健康，头寸就会一直开放。永续期货是MetaFlux上大多数杠杆交易发生的方式。

**订单簿。** 这是某市场中所有人的买卖报价的实时列表，按价格排序。当你的买价与别人的卖价在同一价位相遇时，交易就发生了。*市价单*以当前最优价格成交；*限价单*则在你设定的价格等候。MetaFlux在这些基本类型之上支持许多 [订单类型](concepts/order-types.md)。

**杠杆和保证金。** *保证金*是你用来支撑头寸的抵押品。*杠杆*让该抵押品控制更大的头寸 — 10倍杠杆意味着100美元的存款可以持有1,000美元的头寸。杠杆同时放大收益**和**损失，所以它既强大又风险相等。你的抵押品在头寸之间如何共享或隔离被称为*保证金模式* — 见 [保证金模式](concepts/margin-modes.md)。

**清算。** 如果一个有杠杆的头寸向相反方向变动得足够大，以至于你的抵押品再也无法覆盖它，该头寸会自动平仓以阻止损失继续扩大。这就是*清算*。MetaFlux使用渐进的、 [分级清算](concepts/tiered-liquidation.md) 过程 — 早期警告和分步骤，而不是单一突然的全部损失 — 给头寸恢复的空间。

**资金费率。** 永续期货没有到期日期，所以一个小的定期支付可以保持其价格与真实市场价格挂钩。当更多交易者做多时，多头支付空头；当更多人做空时，空头支付多头。这就是 [资金费率](concepts/funding-rates.md)，它直接在交易者之间支付，而不是支付给交易所。

**标记价格。** MetaFlux不是信任最后一笔交易 — 单个大额或偏离常规的订单可能会扭曲它 — 而是用一个稳健的、抗操纵的参考基准来评估你的头寸，这就是 [标记价格](concepts/mark-prices.md)。它决定了你的保证金、你的清算水平，以及你的未实现盈亏。

:::tip
**简版：** 投入*保证金*，可选地使用*杠杆*扩大规模，对照*标记价格*监控你的头寸健康，避免*清算*。 [词汇表](concepts/glossary.md) 定义了你将遇到的每个术语。
:::

## MetaFlux的独特之处

- **完全在链上且透明。** 每个订单、交易和清算都记录在任何人都可以验证的公共账本上。没有隐藏的撮合，也没有特权的订单簿视图 — 相同的规则适用于每个人。
- **开放且无需许可。** 任何人都可以连接钱包并交易、构建工具和应用程序、或甚至挂牌新市场 — 无需批准。见 [无需许可的市场部署](mip/mip-3.md)。
- **设计具有韧性。** MetaFlux由一套分布式的独立运营者验证，而不是单一公司。没有一个开关可以关闭，没有单点故障，也没有运营者能冻结你的资金。
- **快速。** MetaFlux针对高吞吐量和低延迟进行了优化，所以即使在重负载下交易也能感到响应迅速。

更深层的优势是*能力*，而非价格：MetaFlux投资于先进的市场微观结构、复杂的风险和保证金模型，以及高性能执行层 — 所以平台可以支持更简单场所无法提供的策略和保护。你将在 [概念](concepts/) 部分看到这一点。

## 如何开始

:::info
**只是想交易？** 连接一个支持的钱包，从 [网络和链ID](networks.md) 页面开始，将你的钱包指向正确的网络和端点。
:::

- **选择一个网络。** [网络和链ID](networks.md) 列出了devnet、testnet和mainnet的端点和链ID。如果你想在没有真实资金风险的情况下练习，请从测试网络开始。
- **要构建或运行机器人？** 开发者 [集成快速入门](integration/quickstart.md) 会在几分钟内引导你完成从存款到交易到提现的全过程。
- **选择一个SDK。** [TypeScript](integration/typescript-sdk.md) 和 [Rust](integration/rust-sdk.md) 中现成的客户端为你处理签名和格式转换。
