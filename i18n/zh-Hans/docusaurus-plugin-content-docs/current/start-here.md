# MetaFlux 入门指南

:::info
**初次接触 MetaFlux？** 本页面不预设任何加密货币或衍生品背景知识。读完之后，你将了解 MetaFlux 是什么、目前可以在上面做什么，以及进行第一笔交易前需要掌握的几个关键概念。
:::

## MetaFlux 是什么

MetaFlux 是一个**开放的链上交易所**——一个完全运行在公共网络上、而非某家公司私有服务器内的交易市场。可以把传统交易所想象成一栋有着上锁后台的大楼：你必须信任掌握钥匙的人。MetaFlux 更像一本所有人都能查阅的公共账本，没有任何单一方可以控制它。规则固化在代码中，每笔交易都公开记录在链上，你始终掌握着自己资产的控制权。

正因为运行在链上，MetaFlux 具备**透明性**（任何人都可以验证发生了什么）、**开放性**（任何人都可以接入、在其上构建应用，或上线新市场），以及**持续可用性**（由众多独立运营方共同维护，而非一家可以随时关停的公司）。你只需连接加密钱包、存入资金，即可开始交易——无需开户申请，无需任何门槛。

## 目前你可以做什么

- **交易永续合约。** 通过可选杠杆，对某项资产的价格涨跌进行方向性操作，无需持有资产本身。（"永续"的含义详见下文。）
- **现货交易。** 直接买卖资产，以账户余额结算（仅限余额，暂不支持杠杆）。
- **持有双向（对冲）仓位**——在同一市场同时持有多仓和空仓。详见[对冲模式](concepts/hedge-mode.md)。

即将推出：

- **闲置 USDC 生息**，通过 [Earn](concepts/earn.md) 借贷池获取利息收益。
- **杠杆现货交易**（现货保证金），与 Earn 同期上线。详见[现货保证金](products/spot-margin.md)。

如需查看所有交易产品及其上线状态——包括永续合约、现货、现货保证金，以及规划中的期权和 CDS 方向——请参阅[产品总览](products/index.md)。

## 几个值得了解的核心概念

你不需要成为专家才能开始交易，但以下几个概念能让你更快理解整体逻辑。每个概念都附有更详细的说明链接。

**永续合约 vs. 现货。** *现货*是最直接的交易方式：你用一种资产换取另一种，并持有换来的结果。*永续合约*（"perp"）是一种跟踪资产价格的合约，让你无需持有资产、无需关注到期日，就能从价格的涨跌中获利——只要仓位保持健康，头寸可以一直持有。永续合约是 MetaFlux 上绝大多数杠杆交易的方式。

**订单簿。** 这是某个市场中所有人买卖报价的实时列表，按价格排序。当你的买入报价与某人的卖出报价在相同价格相遇时，交易即成立。*市价单*立即以当前最优价格成交；*限价单*则在你设定的价格处等待成交。MetaFlux 在此基础上支持多种[订单类型](concepts/order-types.md)。

**杠杆与保证金。** *保证金*是你用来支撑仓位的抵押品。*杠杆*让这笔抵押品能够控制更大的仓位——10 倍杠杆意味着 100 美元的存款可以持有 1,000 美元的仓位。杠杆会同等程度地放大盈利**和**亏损，因此既有威力，也伴随风险。你的抵押品如何在不同仓位之间共享或隔离，取决于你的*保证金模式*——详见[保证金模式](concepts/margin-modes.md)。

**清算。** 如果一个杠杆仓位亏损到你的保证金无法继续覆盖，该仓位会被自动平仓以阻止亏损进一步扩大，这就是*清算*。MetaFlux 采用渐进式[分级清算](concepts/tiered-liquidation.md)机制——通过预警提示和分步处理，而非一次性强制平仓——为仓位留出恢复空间。

**资金费率。** 永续合约没有到期日，因此需要通过小额周期性支付来使其价格锚定真实市场价格。当多头持仓者更多时，多头向空头支付；当空头持仓者更多时，空头向多头支付。这就是[资金费率](concepts/funding-rates.md)，费用直接在交易者之间流转，不经过交易所。

**标记价格。** MetaFlux 不以最新成交价作为估值基准——单笔大额或异常订单可能扭曲成交价——而是使用一个稳健、抗操纵的参考价格，即[标记价格](concepts/mark-prices.md)，来对你的仓位进行估值。标记价格决定了你的保证金水平、清算线以及未实现盈亏。

:::tip
**一句话总结：** 存入*保证金*，可选择使用*杠杆*放大仓位，关注仓位在*标记价格*下的健康状态，并避免触发*清算*。[词汇表](concepts/glossary.md)收录了你将遇到的每一个术语。
:::

## MetaFlux 的与众不同之处

- **完全链上，高度透明。** 每一笔订单、交易和清算都记录在任何人都可验证的公共账本上。没有隐藏的撮合机制，没有特权视角——所有人适用同一套规则。
- **开放无许可。** 任何人都可以连接钱包开始交易、在其上开发工具和应用，甚至上线新的交易市场——无需任何审批。详见[无许可市场部署](mip/mip-3.md)。
- **架构上的韧性。** MetaFlux 由一组分布式的独立运营方共同验证，而非由单一公司运营。没有一个可以关停的开关，没有单点故障，也没有哪个运营方能冻结你的资金。
- **高性能。** MetaFlux 针对高吞吐量和低延迟进行了专项优化，即便在高负载下也能保持流畅的交易体验。

MetaFlux 的深层优势在于*能力*，而非价格：MetaFlux 在先进的市场微观结构、精密的风险与保证金模型，以及高性能执行层上持续投入——使平台能够支持其他简单交易场所无法实现的策略和风控保护。你将在[概念](concepts/)章节中深入了解这些内容。

## 如何开始

:::info
**只想直接交易？** 连接支持的钱包，从[网络与链 ID](networks.md) 页面开始，将钱包指向正确的网络和端点。
:::

- **选择网络。** [网络与链 ID](networks.md) 列出了开发网、测试网和主网的端点及其链 ID。如果你想在不承担真实资金风险的情况下练习，建议先从测试网开始。
- **开发者或 Bot 构建者？** 开发者[集成快速入门](integration/quickstart.md)将带你在几分钟内完成从存款、交易到提现的完整流程。
- **选择 SDK。** 现成的 [TypeScript](integration/typescript-sdk.md) 和 [Rust](integration/rust-sdk.md) 客户端已为你处理好签名和通信格式。
