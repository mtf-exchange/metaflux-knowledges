# 产品

MetaFlux 提供的各类**交易产品**。每种产品都是独立的市场类型，拥有各自的订单簿、余额体系和风险模型。本节介绍各产品的定义、当前状态及详细机制文档链接。订单类型、保证金、清算、手续费等各产品共用的底层机制，请参阅[概念](../concepts/index.md)章节。

## 可交易品种

| 产品 | 简介 | 状态 |
|---|---|---|
| [永续合约](./perpetuals.md) | 无到期日的资产价格杠杆做多/做空，通过资金费率锚定 | **已上线** |
| [现货](./spot.md) | 代币对代币 CLOB，结算至账户余额，无杠杆 | **已上线** |
| [现货保证金](./spot-margin.md) | 由 [Earn](../concepts/earn.md) 借贷池提供资金的杠杆现货交易 | **Devnet 预览** |
| [期权](./options.md) | 链上期权（看涨 / 看跌） | **规划中** |
| [CDS](./cds.md) | 信用违约互换式保护合约 | **规划中** |

**永续合约**是默认市场，也是大多数杠杆交易的发生场所——资金费率、标记价格、保证金模式和清算阶梯均以永续合约为默认前提（除非另有说明）。**现货**是仅依赖余额的基础产品，无杠杆。**现货保证金**是现货交易的可选杠杆叠加层，以 [Earn](../concepts/earn.md) 资金池作为借贷供给方。**期权**和 **CDS** 尚在规划阶段，暂无确定的接口协议——详见各自页面的最新进展。

:::info
**仅限无杠杆现货符合伊斯兰教义（Sharia）。** 在上述产品中，只有**无杠杆**[现货](./spot.md)——以全额资产直接买卖，**不使用杠杆、不涉及保证金、不进行借贷、不产生资金费率**——通常被认为符合伊斯兰（Sharia）金融原则。不符合教义的产品明确包括：**现货保证金（杠杆现货交易）**、**永续合约**及所有其他带有杠杆或衍生属性的产品——杠杆和借贷会引入利息（riba）、投机（maysir）和不确定性（gharar）。以上内容仅供参考，不构成宗教或财务建议。
:::
