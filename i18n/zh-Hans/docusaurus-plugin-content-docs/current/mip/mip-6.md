# MIP-6 — 结果市场 / 预测市场

:::info
**推迟至 V3。** 不在 v1 或 v2 范围内。从 MIP-4 重新编号。
:::

MIP-6 是 MetaFlux 的**结果市场 / 预测市场**机制——一种链上市场，用户可在其中交易二元或多元结果的裁定（类似于成熟链上平台上的预测市场改进提案）。这是一项未来能力，推迟至 **V3**。

## 为何单独编号

结果市场**最初编号为 MIP-4**。当项目将 MIP-4 重新定位为[永续合约流动性聚合器 / 内化器](./mip-4.md)后，结果市场概念被重新编号为 **MIP-6**，并从 V2 推入 V3 待办列表。赋予新编号可避免将 MIP-4 用于两个无关机制所带来的混淆。请勿将结果市场称为"MIP-4"。

## 为何推迟

- 这是优先级较低的能力：衍生品 / 永续合约才是主战场，而 MIP-4 聚合器带来的零售收益远超结果市场的机会。
- 结果市场的清算结算引入了核心协议原本无需处理的复杂性：它依赖外部预言机裁定、时间窗口以及争议处理。
- 预测市场在不同司法管辖区存在监管敏感性，待核心协议成熟后再行处理更为稳妥。

结果市场上线时，将以 MIP-6 的形式发布，并配套独立的裁定 / 预言机 / 争议设计——目前均不作预留。
