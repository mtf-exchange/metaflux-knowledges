# MIP-6 — 结果/预测市场

:::info
**推迟至 V3。** 不在 v1 或 v2 范围内。从 MIP-4 重新编号。
:::

MIP-6 是 MetaFlux 的**结果/预测市场**机制 — 链上市场，用户在二元或分类结果的解决上进行交易（类似于已建立的链上场所中的预测市场改进提案）。它是一个未来的差异化轴线，已推迟至 **V3**。

## 为什么以单独编号的形式存在

结果最初编号为 **MIP-4**。当项目将 MIP-4 重新用作[永续流动性聚合器/内化器](./mip-4.md)时，结果概念被重新编号为 **MIP-6** 并从 V2 推移到 V3 积压工作。给它一个全新的编号避免了为两个无关机制重复使用 MIP-4 的混淆。不要将结果称为"MIP-4"。

## 为什么推迟

- 它是较低优先级的差异化轴线：衍生品/永续合约是主要战场，MIP-4 聚合器的零售收入远超结果机会。
- 结果结算增加了核心不需要的清算复杂性：它取决于外部预言机解决、时间窗口和纠纷处理。
- 预测市场具有特定司法管辖区的监管敏感性，最好在核心协议成熟后再处理。

当结果推出时，它作为 MIP-6 推出，具有自己的解决/预言机/纠纷设计 — 这些都不是当今急切保留的。
