# MIP-6 — 结果市场 / 预测市场

:::info
**推迟至 V3。** 不在 v1 或 v2 的范围内。原编号为 MIP-4，现已重新编号。
:::

MIP-6 是 MetaFlux 的**结果市场 / 预测市场**机制——链上市场，用户可在此交易二元或多分类事件的最终结果（类似于成熟链上平台上的预测市场改进提案）。这是未来的差异化方向，已推迟至 **V3** 实现。

## 为什么单独分配编号

结果市场**最初编号为 MIP-4**。当项目将 MIP-4 重新定位为[永续合约流动性聚合器 / 内化器](./mip-4.md)后，结果市场概念被重新编号为 **MIP-6**，并从 V2 推迟至 V3 待办列表。重新分配编号是为了避免 MIP-4 指代两个不相关机制时产生混淆。请勿将结果市场称为"MIP-4"。

## 为什么推迟

- 这是优先级较低的差异化方向：衍生品 / 永续合约才是核心竞争赛道，而 MIP-4 聚合器带来的散户收入远超结果市场的潜在机会。
- 结果市场的结算会引入额外的清算复杂度，而核心协议本身并不需要这些：它依赖外部预言机裁决、时间窗口以及争议处理机制。
- 预测市场在不同司法管辖区存在特定的监管敏感性，待核心协议成熟后再行处理更为稳妥。

结果市场上线时，将以 MIP-6 的形式发布，届时会有独立的裁决 / 预言机 / 争议设计——这些细节目前均不作超前预留。
