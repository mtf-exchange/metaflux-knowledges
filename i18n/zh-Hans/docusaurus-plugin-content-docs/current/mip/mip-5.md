# MIP-5 — 收益存款（Earn）

:::info
**规划中。** MIP-5 将此前预留的插槽分配给 **Earn** —— 一个借贷池，存款人向池中存入资产，并从现货保证金借款人支付的利息中获取收益。具体规范与上线计划将在其所依赖的现货保证金功能完成后跟进。
:::

## 什么是 Earn

Earn 是 **MetaFlux 现货借贷市场的供给侧**。存款人将资产（如 USDC）存入对应的单资产池，获得按池净资产价值（NAV）定价的**份额** —— 其 NAV/份额核算方式与 [Metaliquidity 金库](./mip-2.md) 相同。现货保证金交易者从池中借款并支付利息；利息持续计入池的 NAV，使每份份额的价值不断增长。存款人的收益计算如下：

```
your_yield = shares × (share_price_now − share_price_at_deposit)
```

借款利率是**资金利用率**（已借出 ÷ 总供给）的确定性函数，供给年化收益率（APY）与之联动：

```
supply_APY ≈ borrow_APR × utilisation × (1 − reserve_factor)
```

## 同一市场的两个侧面

Earn（供给侧）与现货保证金（需求侧）共同构成一个完整的借贷市场：借款人支付的利息**即**是贷款人所获得的收益。

- **复用 [MIP-2](./mip-2.md)** 的 NAV/份额核算体系处理存取款（存入时按当前份额价格铸造份额；提取时按 NAV 赎回）。
- 在金库基础上新增了利用率→年化利率的利率曲线，以及逐区块连续计息机制。

## 状态

规划中。依赖现货交易与现货保证金功能先行落地，暂不属于 V1 已完成功能集的一部分。
