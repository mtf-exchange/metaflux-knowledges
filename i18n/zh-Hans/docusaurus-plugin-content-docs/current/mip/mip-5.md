# MIP-5 — 收益

:::info
**计划中。** MIP-5 将之前预留的名额分配给**收益** — 一个借贷池，其中存款人供应资产并从现货保证金借款人支付的利息中获得收益。规范和推出建立在其构建的现货保证金工作之上。
:::

## 什么是收益

收益是 **MetaFlux 现货借贷市场的供应端**。存款人将资产（例如 USDC）借入一个按资产划分的池中，并获得按池的净资产价值 (NAV) 定价的**份额** — 与 [Metaliquidity 金库](./mip-2.md)相同的 NAV/份额会计。现货保证金交易者从池中借款并支付利息；该利息累积进入池的 NAV，因此每份份额升值。存款人的收益为：

```
your_yield = shares × (share_price_now − share_price_at_deposit)
```

借款利率是**利用率**（已借 ÷ 已供应）的确定性函数。供应 APY 追踪它：

```
supply_APY ≈ borrow_APR × utilisation × (1 − reserve_factor)
```

## 一个市场的两端

收益（供应）和现货保证金（需求）是单一借贷市场的两端：借款人支付的利息**就是**贷款人获得的收益。

- **重用 [MIP-2](./mip-2.md)** NAV/份额会计进行存款和取款（存款以当前份额价格铸造份额；取款以 NAV 赎回）。
- 添加金库没有的内容：利用率→APR 利率曲线和连续（按区块）利息累积。
